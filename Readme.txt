# How to push local Repositery to Remote

## This Steps For New Repositery or FirstTime Repositery
 
* git init
* git add README.md
* git commit -m "first commit"
* git branch -M branchName
  git branch -M master
* git remote add originName https://github.com/AzadTom/Git-Github_Leanring.git
  git remote add origin or anyName https://github.com/AzadTom/Git-Github_Leanring.git
* git push -u origin branchName
  git push -u origin master

## Steps to Existing Repositery

* git remote add origin https://github.com/AzadTom/Git-Github_Leanring.git
* git branch -M master
* git push -u origin master
